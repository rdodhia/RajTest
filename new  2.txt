GIT HUB

to create GIT Repository
	 git init
	 
to add files to repo
	git add
	
to check status of repo
	git status
	
to commit file to repo
	git commit -m "Initial project checkin"
	
to check log
	git log
	
to see which repos you have linked
	git remote -v
	
git pull is combination of git fetch and git merge commands.
